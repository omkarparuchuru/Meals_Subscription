# Final Meals

A Flutter application for meal planning and recipes.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- Android Studio / VS Code with Flutter extensions
- iOS development tools (for iOS development on macOS)

### Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   flutter pub get
   ```

### Running the App

To run the app:

```bash
flutter run
```

For a specific device:
```bash
flutter devices
flutter run -d <device_id>
```

### Building the App

**Android:**
```bash
flutter build apk
```

**iOS:**
```bash
flutter build ios
```

**Web:**
```bash
flutter build web
```

## Project Structure

```
lib/
  main.dart          # Main entry point
pubspec.yaml        # Project dependencies and configuration
```

## Development

- Run `flutter analyze` to check for code issues
- Run `flutter test` to run tests
- Run `flutter format .` to format code
